== Description ==

A plugin for adding and customizing syntax highlighting to native PageLines custom code editors, <em>as well as WordPress post, page, & custom post type html editors!</em>

Code Glow offers multiple color schemes to choose from as well as options that control the behavior of the enhanced editors.

All settings are user-specific!
